# Sprint Folders

Quick links to each sprint's documentation folder.

## Naming Conventions

- Folder: `docs/sprints/SXX/`
- Main doc: `README.md` (standardized sprint format)
- Branch: `sprint/SXX-<kebab-description>`
- PR title: `[SXX] category : description`
